# Giao diện Hợp nhất - Unified UI (v4.0.0+)

*(Ngày phát hành: 28/04/2026)*

Chào mừng bạn đến với **Unified UI**, lớp giao diện thế hệ mới của Vnstock. Được thiết kế dựa trên kiến trúc phân tầng (Layered Architecture), Unified UI mang đến một trải nghiệm nhất quán, mạnh mẽ và dễ sử dụng cho cả người dùng mới lẫn các chuyên gia phân tích dữ liệu.

Mục tiêu của Unified UI là chuẩn hóa cách tiếp cận dữ liệu chứng khoán Việt Nam, giúp bạn tập trung vào phân tích thay vì phải bận tâm về cấu trúc phức tạp của các API nguồn.

---

## 📚 Danh mục tài liệu chi tiết

Để khám phá sức mạnh của Unified UI, vui lòng tham khảo các tài liệu chi tiết dưới đây:

1.  **[Tổng quan & Khởi tạo](unified-ui/00-overview.md)**: Tìm hiểu về triết lý thiết kế, cách cài đặt và khởi tạo các domain chính.
2.  **[Tra cứu thông tin (Reference)](unified-ui/01-reference.md)**: Hướng dẫn tìm kiếm cổ phiếu, chỉ số, phái sinh, chứng quyền và thông tin doanh nghiệp.
3.  **[Dữ liệu Thị trường (Market)](unified-ui/02-market.md)**: Cách lấy dữ liệu giá (OHLCV), bảng giá trực tuyến và dữ liệu khớp lệnh (Trades).
4.  **[Phân tích Cơ bản (Fundamental)](unified-ui/03-fundamental.md)**: Truy xuất báo cáo tài chính và các chỉ số định giá.
5.  **[Dữ liệu bán lẻ (Retail)](unified-ui/04-retail.md)**: Theo dõi giá vàng và tỷ giá ngoại tệ.
6.  **[So sánh Miễn phí vs Tài trợ](unified-ui/05-comparison.md)**: Bảng so sánh tính năng 1:1 giữa phiên bản `vnstock` (Miễn phí) và `vnstock_data` (Sponsor).

---

## 🚀 Tại sao nên sử dụng Unified UI?

-   **Tính nhất quán (Consistency)**: Mọi loại tài sản (Cổ phiếu, ETF, Phái sinh,...) đều được truy cập qua một chuẩn lệnh duy nhất.
-   **Tự động gợi ý (Intellisense)**: Hỗ trợ gợi ý code mạnh mẽ trên VS Code, Cursor và Jupyter Notebook.
-   **Độ tin cậy cao**: Tự động chuyển đổi và làm sạch dữ liệu từ nhiều nguồn khác nhau (KBS, VCI, MSN,...) về một định dạng chuẩn.
-   **Dễ dàng nâng cấp**: Bạn có thể chuyển từ phiên bản Miễn phí sang bản Tài trợ mà không cần thay đổi cấu trúc mã nguồn.
